Boyd Bot

A powerful, fast, open-source chatbot helping university students make tasks easier.

Built using Flask

A lightweight WSGI web application framework written in Python enabling developers to easily understand and customize the code.

Extremely Modular

Enables bot to be tailored and changed to your needs.

Open Source

Under the MIT License, the bot is free for use and open for contributions.

These universities use chatbots powered by the Boyd Bot.

University of Glasgow

The Glasgow University Timetable Bot is the first chatbot to be powered by Boyd Bot. This bot is used on Facebook Messenger.
